Crypto

LFSR-线性反馈移位寄存器是一种比较简单的PRNG（伪随机数生成器），属于流密码里的概念。LFSR主要用于生成一串随机的比特流，比特流可用做加密明文的密钥流。

参考https://zhuanlan.zhihu.com/p/33920501

看一下LFSR的基本操作。

已知明文攻击

LFSR存在已知明文攻击，可以通过比特流，枚举度数，然后还原随机的抽头序列。

可以列出下面的方程组，注意异或操作：

101011 001101 0 = P0^P2^P4^P5

1 010110 01101 0 = P1^P3^P4

10 101100 1101 1 = P0^P2^P3

101 011001 101 1 = P1^P2^P5

1010 110011 01 0 = P0^P1^P4^P5

10101 100110 1 1 = P0^P3^P4
